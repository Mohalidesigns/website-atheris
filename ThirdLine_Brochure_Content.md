## DESIGN TIPS FOR ADOBE ILLUSTRATOR

1. **Consistency:** Use the same screenshot frame/device mockup style throughout (e.g., browser window chrome or a custom ThirdLine frame).
2. **Contrast:** Place screenshots on light grey or subtle gradient backgrounds to make them pop against white page areas.
3. **Callout Annotations:** For key screenshots (especially #3, #9, #22), consider adding numbered callout arrows pointing to specific UI elements with brief labels.
4. **Icon Set:** Use a consistent icon family throughout (e.g., Phosphor, Feather, or custom). The selling points grid on Page 3 is the most icon-heavy section.
5. **White Space:** Let the content breathe — this is a premium enterprise product. Avoid cramming.
6. **Bleed & Margins:** If printing, set up 3mm bleed and 10mm safety margins on all pages.
7. **Export Settings:** Final PDF at 300 DPI, CMYK colour mode for print, or RGB for digital distribution.
